# Exercício 1 — Classe ProdutoTecnologia

Crie uma classe chamada **ProdutoTecnologia** que represente um produto vendido em uma loja de eletrônicos.

A classe deve possuir os seguintes atributos **privados**:

- nomeProduto
- marca
- preco
- quantidadeEstoque

Crie:

- Um **construtor** que receba todos os atributos
- Métodos **getters e setters**
- Um método chamado **exibirProduto()** que mostre todas as informações

No `main`:

1. Crie **dois objetos**
2. Exiba os dados dos produtos
